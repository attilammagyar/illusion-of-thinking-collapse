Let’s solve the classic **Tower of Hanoi** puzzle for **12 disks**.

## Problem Restatement

- **Pegs:** 0 (start), 1 (auxiliary), 2 (goal)
- **Disks:** 1 (smallest/top) to 12 (largest/bottom)  
- **Goal:** Move all disks from peg 0 to peg 2, obeying the rules.

## Solution Approach

The solution uses **recursion**:

- Move the top \( n-1 \) disks from peg 0 to peg 1 (using peg 2 as auxiliary).
- Move disk \( n \) from peg 0 to peg 2.
- Move the \( n-1 \) disks from peg 1 to peg 2 (using peg 0 as auxiliary).

Let’s write a function to generate the moves.

---

### Python Function to Generate Moves

```python
def hanoi(n, source, target, auxiliary, moves):
    if n == 0:
        return
    hanoi(n-1, source, auxiliary, target, moves)
    moves.append([n, source, target])
    hanoi(n-1, auxiliary, target, source, moves)

moves = []
hanoi(12, 0, 2, 1, moves)
```
