Let's solve the **Tower of Hanoi** puzzle for **8 disks** using the recursive algorithm.  
We'll output the **complete list of moves** in the required format:

```
moves = [[disk id, from peg, to peg], ...]
```

---

### **Algorithm Explanation**

For `n` disks from `from_peg` to `to_peg` using `aux_peg`:
1. Move `n-1` disks from `from_peg` to `aux_peg` using `to_peg` as auxiliary.
2. Move disk `n` from `from_peg` to `to_peg`.
3. Move `n-1` disks from `aux_peg` to `to_peg` using `from_peg` as auxiliary.

---

### **Python Implementation (for reference)**

```python
def hanoi(n, from_peg, to_peg, aux_peg, moves):
    if n == 0:
        return
    hanoi(n-1, from_peg, aux_peg, to_peg, moves)
    moves.append([n, from_peg, to_peg])
    hanoi(n-1, aux_peg, to_peg, from_peg, moves)

moves = []
hanoi(8, 0, 2, 1, moves)
print(moves)
```
