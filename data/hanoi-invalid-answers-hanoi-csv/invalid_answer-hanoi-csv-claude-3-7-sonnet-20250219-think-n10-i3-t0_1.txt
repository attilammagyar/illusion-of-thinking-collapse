```python
    if n > 0:
```

The Tower of Hanoi is a classic problem with a well-known recursive solution. For n=10, we need 2^10 - 1 = 1,023 moves.

Let me generate the complete list of moves using the recursive algorithm:

```python
def hanoi_moves(n, source, auxiliary, target):
    if n == 1:
        return [(1, source, target)]
    
    moves = hanoi_moves(n-1, source, target, auxiliary)
    moves.append((n, source, target))
    moves.extend(hanoi_moves(n-1, auxiliary, source, target))
    
    return moves

moves = hanoi_moves(10, 0, 1, 2)
for disk, from_peg, to_peg in moves:
    print(f"{disk},{from_peg},{to_peg}")
```
