Of course. This is a classic puzzle known as the Tower of Hanoi. The solution can be found using a recursive algorithm.

### The Strategy

To move a stack of $n$ disks from a `source` peg to a `destination` peg using an `auxiliary` peg, we follow these steps:

1.  Move the top $n-1$ disks from the `source` peg to the `auxiliary` peg.
2.  Move the single remaining disk (the largest, disk $n$) from the `source` peg to the `destination` peg.
3.  Move the $n-1$ disks from the `auxiliary` peg to the `destination` peg.

The base case for the recursion is moving a single disk ($n=1$), which is a simple one-step move.

For your puzzle with 12 disks, we need to move them from peg 0 to peg 2, using peg 1 as the auxiliary peg. This will require $2^{12} - 1 = 4096 - 1 = 4095$ moves.

### Exploration with a 2-Disk Example

Let's trace the logic for $n=2$ disks from peg 0 to peg 2.

- **Initial State**: `[[2, 1], [], []]`
- **Goal**: Move 2 disks from peg 0 to peg 2.

1.  **Move $n-1=1$ disk from source (0) to auxiliary (1).**
    - This is a base case. Move disk 1 from 0 to 1.
    - `moves = [[1, 0, 1]]`
    - State: `[[2], [1], []]`
2.  **Move disk $n=2$ from source (0) to destination (2).**
    - Move disk 2 from 0 to 2.
    - `moves = [[1, 0, 1], [2, 0, 2]]`
    - State: `[[], [1], [2]]`
3.  **Move $n-1=1$ disk from auxiliary (1) to destination (2).**
    - This is a base case. Move disk 1 from 1 to 2.
    - `moves = [[1, 0, 1], [2, 0, 2], [1, 1, 2]]`
    - State: `[[], [], [2, 1]]` (Goal reached)

The complete list of moves for $n=2$ is `[[1, 0, 1], [2, 0, 2], [1, 1, 2]]`.
